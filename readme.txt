	Guide to uplink the website.

1. Install xampp using the xampp installation guide attached.
2. Place the campus folder inside the /Xampp/htdocs/
3. go to the /Xampp/htdocs/campus/db/config.php directory
4. give username password of the data base over here.
5. Start XAMPP using xampp control panel, or start.exe in xampp directory.
6. go to your localhost/phpMyadmin and place the install.sql file content in the sql tab.
7. Now your project is up and running at the URL localhost/campus/index.php.

*. Make sure internet connectivity is there if using mailing module, else no need of internet connectivity.
